## 🌟 Overview

The **Personal Notes App** is a lightweight and user-friendly web application that helps users save important thoughts, ideas, reminders, and daily notes.

All notes are stored using **Local Storage**, ensuring they remain available even after refreshing or reopening the browser.

## ⚙️ How It Works

### 1️⃣ Write a Note

Enter your thoughts in the text area.

### 2️⃣ Save

Click the **Save Note** button.

### 3️⃣ Store

The note is stored in the browser using Local Storage.

### 4️⃣ View

Saved notes appear instantly below the form.

### 5️⃣ Delete

Remove notes anytime with a single click.
